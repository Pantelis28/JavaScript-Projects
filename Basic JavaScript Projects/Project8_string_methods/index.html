The concat() method joins two or more strings.

The slice() method selects from a given start, up to a (not inclusive) given end.

The toUpperCase() method converts a string to uppercase letters.

hello world!

The search() method returns the index (position) of the first match.

The toString() method can be used to convert a string object into a string.

The toPrecision() method formats a number to a specified length.

toFixed() converts a number to a string, rounded to a specified number of decimals:

5.56789

The valueof() returns the primitive value of an integer
